# Workout Tracker

A full-featured workout tracking application for recomposition athletes. Track training, body composition, nutrition, and progress in one place.

## Features

- **Train** — Daily dashboard, workout logger with RPE, rest timer, fatigue check-in, superset grouping
- **Body** — Daily weigh-in, weight chart (Chart.js), body measurements, 12-week adherence grid, gym cost tracker
- **Fuel** — Nutrition logging, macro donut chart, water intake tracker, calorie burn sync
- **Progress** — PR tracker, strength standards vs bodyweight, plateau detection, weekly review, custom program loader

## Tech

Vanilla HTML/CSS/JS. Chart.js for visualizations. localStorage for data persistence.

## Usage

Open `src/index.html` in a browser or deploy via Vercel.
